# Ashtakavarga Reference PDFs

Place your Ashtakavarga reference books here. The Ashtakavarga analysis mode reads ALL PDFs in this directory as its sole authoritative source.

## Recommended texts

- **BV Raman — "Ashtakavarga System of Prediction"** — the most widely used standalone Ashtakavarga reference
- **BPHS Chapters on Ashtakavarga** (Brihat Parashara Hora Shastra) — classical foundation
- **CS Patel — "Ashtakavarga"** — detailed computational examples
- Any other Ashtakavarga text you trust and want the analysis grounded in

## How it works

When you trigger Ashtakavarga mode ("run Ashtakavarga analysis"), the skill will:
1. Read every PDF in this folder
2. Use the methodology described in these PDFs as the exclusive basis for analysis
3. Not mix in Dasha, Yoga, or other Parashari/Nadi methods (unless you ask)

The more thorough your reference PDFs, the more precise the analysis will be.
